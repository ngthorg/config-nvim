# My config Nvim

```bash
git clone https://github.com/ngthorg/config-nvim.git ~/.config/nvim
```

Next we need to install python support (node is optional)

- Neovim python support

  ```bash
  pip install pynvim
  ```

- Neovim node support

  ```bash
  npm i -g neovim
  ```
